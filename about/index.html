About ZER0-Nu1L

ZERO

神作《叛逆的鲁路修》主人公鲁路修·Vi·不列颠尼亚对外奉行边沁提出的功利主义，企图用尽手段得到推翻弱肉强食的社会达尔文式统治的结果。毅然化身 ZERO，一次次用自己的智慧、野心与手段冲击统治阶层，最后以一句“我毁灭了世界，也创造了世界”作为一生的悲壮结语。
自始至终，鲁路修没有被野心以及王之意志支配，没有复演暴君理论。

ZERO 意味着鲁路修一次次堕入失落和绝望但又一次次坚定，ZERO 意味着要将扭曲的世界归于零，然后从零重新建立新的温柔的世界的决心；
ZERO 不仅仅表示一个人，也代表一个标志。一个没有在黑压压的世界里睡去的灵魂，一种敢于革命、敢于和绝对权威对抗的高傲，一种颠覆所有人认知的存在。

笔者崇尚康德的绝对主义，但是这样理想的我总是在现实生活中处处碰壁，同时也意识到这样也会禁锢自己。而《叛逆的鲁路修》给我的启示是只有联合枢木朱雀和鲁路修（绝对主义和功利主义），才成为最后的 ZERO，才能真正地改变世界。
投资自己的信念，依然想要完全保持绝对主义的清高实在困难，投资就意味着风险。

不要失去狂妄的梦想，世界没有救世主，如果每个人都等着别人来改变世界，那么世界就不会被改变，只要自己开始行动，世界就在被改变；不要被反噬，心中一定要永存一朵觉得不可以糜烂、不可以肮脏、不可以堕落、不可以腐败、不可以低级趣味的花；不要自我怀疑，崩塌了就重建，失败了就重头再来，信念犹在，志业犹在，安身立命之本犹在，何尝失落？

以此致敬《叛逆的鲁路修》。

NULL

NULL 在 C 语言中表示空指针。学习 C 语言是我们认识计算机的开端，很多高级语言不直接对用户开放指针操作，但是指针确实是理解计算机内部组成的一个引子。一个指针可以直接从一个变量的根源上修改它，哪怕它们不在一个函数体内。数据结构大多也是靠指针来实现。每个变量之间看似独立，但是又总是有指针将它们联系起来。

每个人都是一座孤岛。但是他们又在某个层面相互联系。
我曾经觉得自己是个空指针 NULL，什么都无法拯救、什么都无法改变，甚至觉得自己是个封闭系统，固执地守护着自己的价值观，不会被人改变。

终于有一天看到那些自己无意识播种下的，在自己看得见的地方和看不见的地方生根发芽，发挥着作用，而它们也无意识地反作用于我。
我没有想到自己当初日日夜夜的前行会激励某个在黑暗中前行的插班生，我也没有想到再也见不到的她也在精神上鼓励没有融入新环境的我；我没有想到自己拙劣的文字很多时候会打动别人，维系着他们的生活；没有想到过去的自己留下的痕迹会深深地影响着自己……

我们觉得我们是一座无人问津的孤岛，觉得是一个不被指向也不指向谁的空指针 NULL。
但是事实上我们或许是指向人类的高阶指针，所有的所有都在看得见和看不见的地方发生着作用。

润物细无声。

ZERO & NULL

ZERO和NULL 都表示空。以空来命名，其实也想表达生命本来没有名字的哲学。
周国平先生的感悟对我极具启示性，私以为将他的文字引过来不加赘述是这部分最好的阐述方式。

“生命本来没有名字”——这话说得多么好!我们降生到世上，有谁是带着名字来的?又有谁是带着头衔、职位、身份、财产等等来的?可是，随着我们长大，越来越深地沉溺于俗务琐事，已经很少有人能记起这个最单纯的事实了。我们彼此以名字相见，名字又与头衔、身份 、财产之类相连，结果，在这些寄生物的缠绕之下，生命本身隐匿了，甚至萎缩了。无论对己对人，生命的感觉都日趋麻痹。多数时候，我们只是作为一个称谓活在世上。即使是朝夕相处的伴侣，也难得以生命的本然状态相待，更多的是一种伦常和习惯。浩瀚宇宙间，也许只有我们的星球开出了生命的花朵，可是，在这个幸运的星球上，比比皆是利益的交换，身份的较量，财产的争夺，最罕见的偏偏是生命与生命的相遇。仔细想想，我们是怎样地本末倒置，因小失大，辜负了造化的宠爱。
是的 ——我是，你是，每一个人都是一个多么普通又多么独特的生命，原本无名无姓，却到 底可歌可泣。我、你，每一个生命都是那么偶然地来到这个世界上，完全可能不降生，却毕竟降生了，然后又将必然地离去。想一想世界在时间和空间上的无限，每一个生命的诞生的 偶然，怎能不感到一个生命与另一个生命的相遇是一种奇迹呢。有时我甚至觉得，两个生命 在世上同时存在过，哪怕永不相遇，其中也仍然有一种令人感动的因缘。我相信，对于生命的这种珍惜和体悟乃是一切人间之爱的至深的源泉。你说你爱你的妻子，可是，如果你不是把她当作一个独一无二的生命来爱，那么你的爱还是比较有限。你爱她的美丽、温柔、贤惠 、聪明，当然都对，但这些品质在别的女人身上也能找到。惟独她的生命，作为一个生命体的她，却是在普天下的女人身上也无法重组或再生的，一旦失去，便是不可挽回地失去了。 世上什么都能重复，恋爱可以再谈，配偶可以另择，身份可以炮制，钱财可以重挣，甚至历 史也可以重演，惟独生命不能。愈是精微的事物愈不可重复，所以，与每一个既普通又独特 的生命相比，包括名声地位财产在内的种种外在遭遇实在粗浅得很。
既然如此，当另一个生命，一个陌生得连名字也不知道的生命，远远地却又那么亲近地发现了你的生命，透过世俗功利和文化的外观，向你的生命发出了不求回报的呼应，这岂非人生中令人感动的幸遇?

ZER0-Nu1L

ZER0-Nu1L 中将 O 换成了 0, 将 L 换成了 l（也有一些大小写的切换）。
这其实是编写程序学习阶段容易遇到的障眼法。同时1,0也是最原本的机器语言的组成部分。
个人的私心是将自己对 ZERO、NULL浅薄的理解，融入到自己将要从事的领域。

网络安全对于我来说，是一种“守护”的人生信仰。（也算是受到《死神》、《排球》少年的影响）
笔者没有什么壮志雄心，只想守护好自己所爱的、自己所信奉的，亲人、社会、互联网世界、国家、人类、自我的信仰。
也相信自己的微薄的努力会在看得见和看不见的地方发生着作用。
